### Camera Implementation

```
startCamera() flow:
1. Stop any existing stream (cameraStreamRef.current?.getTracks().forEach(t => t.stop()))
2. getUserMedia({ video: { width: { ideal: 640 }, height: { ideal: 480 }, facingMode: 'user' }, audio: false })
3. await new Promise(resolve => setTimeout(resolve, 100))  ← wait for DOM
4. videoRef.current.srcObject = stream
5. videoRef.current.muted = true
6. videoRef.current.playsInline = true
7. await new Promise((resolve, reject) => {
     videoRef.current.onloadedmetadata = () => videoRef.current.play().then(resolve).catch(reject)
     setTimeout(reject, 10000)
   })
8. setCameraStatus('active')
```

Handle errors: `NotAllowedError`, `NotFoundError`, `NotReadableError`.

### Face Tracking — MediaPipe FaceMesh (CDN, no npm install)

Load dynamically:
```javascript
const loadMediaPipe = () => new Promise((resolve, reject) => {
  if (window.FaceMesh) { resolve(); return; }
  const s1 = document.createElement('script');
  s1.src = 'https://cdn.jsdelivr.net/npm/@mediapipe/camera_utils/camera_utils.js';
  s1.crossOrigin = 'anonymous';
  const s2 = document.createElement('script');
  s2.src = 'https://cdn.jsdelivr.net/npm/@mediapipe/face_mesh/face_mesh.js';
  s2.crossOrigin = 'anonymous';
  s1.onload = () => document.head.appendChild(s2);
  s2.onload = resolve;
  s2.onerror = () => reject(new Error('MediaPipe load failed'));
  s1.onerror = () => reject(new Error('Camera utils load failed'));
  document.head.appendChild(s1);
});
```

FaceMesh config: `maxNumFaces: 2`, `refineLandmarks: true`, `minDetectionConfidence: 0.5`

Per-frame analysis:
- **Eye Aspect Ratio (EAR):** landmarks `[33,160,158,133,153,144]` (left) and `[362,385,387,263,373,380]` (right). EAR > 0.20 = eyes open.
- **Head pose:** yaw = `(nose.x - faceCenter.x) / faceWidth`. `|yaw| < 0.25` = looking at screen.
- **Expression:** mouth open ratio (landmarks 13, 14 > 0.02 = speaking). Brow raise (landmark 70 vs 33 > 0.04 = attentive).
- **Attention score:** `(lookingAtScreen ? 60 : 20) + (eyesOpen ? 30 : 0) + (singleFace ? 10 : 0)`

Canvas drawing per frame:
- Green bounding box around first face, red around extras
- Blue eye outlines (left + right)
- Key landmark dots at indices `[10, 152, 234, 454, 33, 263, 1, 61, 291, 199]`
- "● You" label at first face, "● Extra Person" at others
- Red dashed border + "No Face Detected" text when no face found

Fallback if MediaPipe CDN fails: pixel diff motion detection on canvas. No crash, just a "Basic detection active" warning badge.

### Microphone Test

State machine: `idle → requesting → testing → pass | no_audio | fail`

- **Triggered by button click** (never auto) — this is the user gesture that unlocks `AudioContext`
- `getUserMedia({ audio: { echoCancellation: true, noiseSuppression: true }, video: false })`
- Create `AudioContext` AFTER `getUserMedia` resolves
- `AnalyserNode` → `fftSize: 512`, RMS over `Uint8Array`
- Sample for 3 seconds, show live animated level bar (20 bars, height driven by `micLevel` 0–100)
- Pass if more than 5 samples above threshold 5
- `no_audio` state if connected but silent — show "Make sure mic is not muted" + Retry
- Stop stream + close AudioContext after test

### Speaker Test

State machine: `idle → playing → confirm → pass | fail`

- **Triggered by button click** — same user gesture rule
- Create `AudioContext` directly inside click handler
- Play 3-tone chime: C5 (523.25 Hz), E5 (659.25 Hz), G5 (783.99 Hz) with sine oscillators and smooth gain ramps
- After chime: show "Did you hear the chime?" with Yes / No buttons
- Yes → `pass`. No → `fail` with "Check volume and output device" tip

## BACKEND — `/interview/answer` CODE SUBMISSION

The backend endpoint must handle code answers. When the answer starts with `[Code:`, extract the language and code, and include them in the Groq prompt differently:

```python
# In backend/routers/interview.py, inside the answer handler:

is_code_answer = answer.strip().startswith('[Code:')

if is_code_answer:
    # Parse: "[Code: python]\n\nactual_code_here"
    lines = answer.split('\n', 2)
    lang_line = lines[0]  # "[Code: python]"
    code_body = lines[2] if len(lines) > 2 else answer
    language = lang_line.replace('[Code:', '').replace(']', '').strip()
    
    user_content = f"""The candidate submitted code in {language}:

```{language}
{code_body}
```

Evaluate this code: Is the logic correct? Is it efficient? What is the time/space complexity? 
Ask a follow-up question about the code or move to the next question."""
else:
    user_content = answer
```

The AI then evaluates the code and responds naturally. No code execution on the backend is needed — the AI reviews it conversationally.
